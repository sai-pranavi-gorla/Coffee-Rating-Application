# Coffee Rating Application

## Description

A web application that displays a list of coffee items and allows users to vote for their favorite coffee using a button. Each vote updates the vote count stored in an SQLite database and displays the updated result on the page.

## Technologies Used

* HTML
* CSS
* Python (Flask)
* SQLite

## Features

* Display coffee items
* Vote using a button
* Update vote count in database
* Show updated votes instantly
